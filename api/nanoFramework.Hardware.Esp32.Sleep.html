    StartDeepSleep()

    Enter deep sleep using configured wakeup sources.

    Declaration
    public static void StartDeepSleep()
    Remarks

    If no wakeup sources configured then it will be a indefinite sleep. This call never returns. After the device enters deep sleep a wakeup source will wake the device and the execution will start as if it was a reset. Keep in mind that the execution WILL NOT continue after the call to this method.
